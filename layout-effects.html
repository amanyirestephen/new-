Computer Science

What is an Algorithm?

An algorithm is a finite, well-defined set of instructions for solving a computational problem. Every program you write is a collection of algorithms.

Deep Dive

Algorithms are evaluated on two axes: time complexity (how fast?) and space complexity (how much memory?). Big-O notation gives us a language to compare them regardless of hardware.

Common complexities: O(1) constant, O(log n) logarithmic, O(n) linear, O(n²) quadratic. Always aim for the lowest you can achieve.
